# LLM Data Analīzes Projekts

## Apraksts
Šis projekts izmanto LLM, lai:
- izveidotu datu analīzes plānu
- ģenerētu SQL vaicājumus
- veidotu datu vizualizācijas
- izveidotu HTML atskaiti

## Rezultāts
Atver `report.html`, lai redzētu gala rezultātu.

## Tehnoloģijas
- Python
- Matplotlib
- MySQL (simulēts)
- LLM Gemini
